# MecFem

Finite element solver for mechanics

## Installation

### Project Structure

- `src/MecFEM/` - Core finite element library
- `app/` - GUI application built with Flet
- `test/` - Some tests for MecFEM library

### Clone the repository

```bash
git clone https://github.com/GabrielJezler/MecFem.git
cd MecFem
```

### Run flet app

- Install MecFEM as editable package

```bash
pip install -e .
```

- Run the app

```bash
cd app
flet run
```
